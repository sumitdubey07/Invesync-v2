# Invesync — Paper Trading Platform for Indian Stocks

Practice trading NSE stocks with ₹1,00,000 virtual money.
Real market data, zero financial risk.

## Live Demo
🔗Website: https://invesync-client.vercel.app

🔗Backend: https://invesync-server.onrender.com

## Features
- 📈 Real-time NSE/BSE stock prices (Yahoo Finance)
- 💰 ₹1,00,000 virtual balance on signup
- 🛒 Buy/sell paper trades with market orders
- 📊 Portfolio tracker with live P&L
- 🕯️ Candlestick + line charts (1D/1W/1M/3M/1Y)
- 🤖 AI price prediction (linear regression, 7-day forecast)
- 📰 Live stock news feed
- 🏆 Leaderboard — compete with other traders
- 🥧 Sector exposure breakdown chart
- 📝 Trade notes
- 📱 Mobile responsive with bottom tab bar

## Tech Stack
### Frontend
- React + Vite
- Tailwind CSS
- Recharts + Lightweight Charts
- React Router v6
- Axios

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Yahoo Finance API (proxied)
- Finnhub API (news)

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
